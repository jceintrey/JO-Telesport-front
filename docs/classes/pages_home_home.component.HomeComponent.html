Class HomeComponent

Composant d'accueil de l'application. Ce composant affiche un aperçu des données olympiques, y compris le nombre de pays participants et le nombre total de Jeux Olympiques enregistrés.

  • Utilise le service OlympicService pour récupérer les données.
  • Gère les graphiques via ngx-charts.
